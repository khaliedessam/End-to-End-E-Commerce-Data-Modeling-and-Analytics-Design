## Project Overview
This project demonstrates the complete engineering lifecycle of an enterprise-grade data platform. It spans from initial business process analysis and relational database design to the implementation of a modern analytical Data Warehouse utilizing the Medallion Architecture.

The solution covers robust entity-relationship modeling, strict normalization, automated ETL/ELT pipeline orchestration via SQL Server Agent, and an analytical reporting layer designed for high-performance business intelligence.
